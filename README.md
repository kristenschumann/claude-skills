# claude-skills

5 shareable Claude Code skills for working smarter in any project.

## Install

```bash
npx skills add kristenschumann/claude-skills --skill learn
npx skills add kristenschumann/claude-skills --skill scope
npx skills add kristenschumann/claude-skills --skill time-audit
npx skills add kristenschumann/claude-skills --skill executive-critique
npx skills add kristenschumann/claude-skills --skill kpi-bot
```

## Skills

### /learn
Post-session retrospective. Audits what went well and what didn't, proposes updates to skills, memory, and CLAUDE.md, and gives coaching feedback on how to work with Claude more efficiently.

### /scope
Declare a scope mode before planning: EXPAND (generate ideas), HOLD (refine only), or REDUCE (cut ruthlessly). Monitors for scope drift and maintains a parked ideas list.

### /time-audit
Weekly time audit using macOS Screen Time and Chrome history. Outputs an HTML visualization showing how you actually spent your time vs. how you intended to.

### /executive-critique
Pressure-test a plan through two executive lenses: operational (execution, timelines, dependencies) and strategic (ROI, ambition, customer psychology). Surfaces tension between perspectives.

### /kpi-bot
Builds a daily Slack KPI bot. Describe the metric you want to track, and it writes the SQL, formats it for Slack, and creates or updates an n8n workflow to post it automatically.
